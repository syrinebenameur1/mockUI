## ðŸ›  Technical Architecture

### Component Structure
*   **`App.jsx`**: The main container. It handles the sidebar navigation and conditionally renders the active Module.
*   **`src/modules/`**: Contains the three main route components:
    *   `Module1_AssessmentsMatching.jsx` (Green)
    *   `Module2_Communication.jsx` (Red)
    *   `Module3_Operations.jsx` (Blue)
*   **`src/shared/DataContext.jsx`**: A React Context provider that holds the "Global State" (mock database). It passes down data like `projects`, `developers`, `invoices`, and `messages` to all modules, simulating a real connected backend.

### Styling
*   **`index.css`**: Contains global CSS variables (colors, fonts, spacing) and utility classes (`.card`, `.btn`, `.badge`, `.table-simple`).
*   **Theme System**: The app implements a rigorous "Light Professional" theme using Slate grays and distinct brand colors (Green/Red/Blue) for each module.

### Icons
*   **Lucide React**: Used for all iconography to ensure a consistent, clean, and professional look.
